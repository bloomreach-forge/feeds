Introduction

What is the Feed Creator?

Feed Creator is a plugin for Bloomreach Experience Manager, f.k.a. Hippo CMS, that lets you create RSS feeds and ATOM feeds. It provides an API that enables you to create your own web syndications with the CMS, using documents. You can create as many feeds as you like, and for each feed you can setup filters to determine which documents to include.

Code base on GitHub

The code of this project is on GitHub at https://github.com/bloomreach-forge/feeds/.
